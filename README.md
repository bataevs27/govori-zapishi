# Govori-Zapishi — установка

Автоматическая расшифровка созвонов с разделением по спикерам. Работает на Mac с Apple Silicon (M1/M2/M3/M4).

**Модели:**
- 🎙 [MLX Whisper medium](https://github.com/ml-explore/mlx-examples) — транскрипция речи, запускается на Neural Engine / GPU через Apple MLX
- 👥 [pyannote.audio 3.1](https://github.com/pyannote/pyannote-audio) — диаризация спикеров (кто что сказал), запускается на GPU через MPS

---

## Шаг 1. HuggingFace — аккаунт, лицензии, токен

Нужен для модели диаризации.

### 1.1 Создайте аккаунт
Зарегистрируйтесь на https://huggingface.co

### 1.2 Примите лицензии
Войдите в аккаунт. Откройте каждую ссылку и нажмите **"Agree and access repository"**:

- https://huggingface.co/pyannote/speaker-diarization-3.1
- https://huggingface.co/pyannote/segmentation-3.0
- https://huggingface.co/pyannote/speaker-diarization-community-1

### 1.3 Создайте токен
1. Перейдите: https://huggingface.co/settings/tokens
2. **"New token"** → название любое → тип **Read** → **"Generate a token"**
3. Скопируйте токен — вставите его в приложение при первом запуске

---

## Шаг 2. Установка и запуск

Перетащите `GovoriZapishi.app` в папку **Программы (Applications)**. Откройте через Launchpad или Spotlight.

Если появилась ошибка **"не удаётся открыть"**:
1. Откройте **Системные настройки → Конфиденциальность и безопасность**
2. Прокрутите вниз → **"Всё равно открыть"**

**При первом запуске приложение само установит всё необходимое:**
- Если нет Homebrew — откроется Terminal с командой установки. После завершения выполните команды из раздела «Next steps» и перезапустите GovoriZapishi.
- Python, ffmpeg, BlackHole и пакеты установятся автоматически (~10-15 мин). Придут уведомления о прогрессе.

### После запуска

Приложение спросит две папки — для транскриптов встреч и для заметок (можно папку в Obsidian vault).

Затем откройте **⏺ → ⚙️ Настройки**:
1. Вставьте токен из шага 1.3, нажмите **"Сохранить токен"**
2. Нажмите **"Проверить статус"** — напротив каждой лицензии должен быть ✅
3. Если ❌ — нажмите **"Открыть →"** рядом с ней, примите лицензию, проверьте снова

Подождите пока статус сменится на **"Готово"** — при первом запуске скачиваются модели (~1 ГБ).

## Шаг 3. Настройка Audio MIDI Setup

Откройте: **Finder → Программы → Утилиты → Audio MIDI Setup**

### 3.1 Multi-Output Device
Нажмите `+` → **Create Multi-Output Device**

- ✅ **BlackHole 2ch**
- ✅ **Ваши наушники или MacBook Pro Speakers**

Primary Device: **BlackHole 2ch**

### 3.2 RecordInput
Нажмите `+` → **Create Aggregate Device**, назовите **RecordInput**

- ✅ **Ваш микрофон** — Clock Source
- ✅ **BlackHole 2ch** — Drift Correction

Clock Source: **ваш микрофон**

|---|---|

## Использование

**Встреча:**
- Перед созвоном: **⏺ → 🤝 Записать встречу**
- После созвона: **⏺ → ⏹ Стоп встречи**
- Файл `.md` появится в папке встреч

**Заметка:**
- **⏺ → 📝 Записать заметку** → **⏹ Стоп заметки**

Можно записывать новую встречу или заметку пока идёт обработка предыдущей — они встанут в очередь.

## Возможности v1.0.0

- 🤝 **Запись встреч** — микрофон + системный звук, транскрипция с разделением по спикерам (`SPEAKER_00`, `SPEAKER_01`, ...)
- 📝 **Голосовые заметки** — только микрофон, транскрипция без диаризации, обрабатывается быстрее
- 📋 **Очередь обработки** — записывайте следующую встречу не дожидаясь конца обработки предыдущей
- ⚡️ **Приоритет заметок** — заметки обрабатываются первыми в очереди
- ⏱ **Таймер обработки** — в меню-баре отображается обратный отсчёт до конца обработки
- 📁 **Раздельные папки** — встречи и заметки сохраняются в разные папки (настраивается)
- 🗑 **Автоудаление аудио** — аудиозаписи хранятся 7 дней и удаляются автоматически если транскрипт создан
- ⚙️ **Окно настроек** — смена папок, управление токеном, проверка статуса лицензий HuggingFace
- 🚀 **Автоустановка** — все зависимости устанавливаются автоматически при первом запуске

## Формат транскрипции встречи

```markdown
# Встреча 24.04.2026, 11:30

**[11:30:00 — 11:30:05] SPEAKER_00:** Добрый день, начнём?
**[11:30:06 — 11:30:12] SPEAKER_01:** Да, всем привет.
```
